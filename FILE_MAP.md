# OpenClaw WSL File Map

- **openclaw.json**: `~/.openclaw/openclaw.json` (Configuration file containing model, API keys, and system settings).
- **sessions/**: `~/.openclaw/agents/main/sessions/` (Stores active channel connections like WhatsApp sessions).
- **workspace/**: `~/.openclaw/workspace/` (Default directory where OpenClaw agents read and write temporary project files).
